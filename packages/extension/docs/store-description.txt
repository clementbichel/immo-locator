DESCRIPTION COURTE
------------------
L'adresse exacte d'un bien Leboncoin ou SeLoger, retrouvée en un clic. Gratuit.


DESCRIPTION LONGUE
------------------
🔍 TROUVEZ L'ADRESSE EXACTE. AVANT MÊME DE CONTACTER LE VENDEUR.

Les annonces Leboncoin et SeLoger n'affichent pas l'adresse du bien. Résultat : vous contactez, vous attendez, vous vous déplacez — parfois pour rien. Immo Locator change ça.

Ouvrez une annonce immobilière, cliquez sur l'icône, et l'extension fait le reste.

⚡ CE QU'ELLE EXTRAIT INSTANTANÉMENT

📍 Adresse exacte du bien + lien Google Maps direct
📐 Surface habitable et terrain
🏷️ Étiquettes énergie (DPE / GES) et consommations réelles
🏙️ Ville et code postal

L'adresse est retrouvée en croisant les données de l'annonce avec la base publique officielle des diagnostics énergétiques de l'ADEME (ministère de la Transition écologique).

🏠 POUR LES ACHETEURS COMME POUR LES INVESTISSEURS

Que vous cherchiez votre future maison ou que vous analysiez des dizaines d'annonces, Immo Locator vous permet de vérifier l'emplacement exact, croiser les données énergétiques et éliminer les mauvaises surprises — sans perdre de temps.

🔒 VIE PRIVÉE PAR CONCEPTION

✅ Aucun compte requis
✅ Aucune donnée personnelle collectée
✅ Aucun serveur intermédiaire : l'extension interroge directement la base publique de l'ADEME. Vos recherches ne transitent par aucun serveur tiers.
✅ L'extension ne lit que les pages d'annonces Leboncoin et SeLoger — rien d'autre

🎉 100% gratuit. Instantané. Indispensable pour votre recherche immobilière.
