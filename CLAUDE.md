## Project Overview

This is a Chrome Extension (Manifest V3) for extracting real estate data from Leboncoin listings and cross-referencing them with an official energy performance database via a backend API.

**Purpose**: Extract property details (surface, DPE, GES, location, energy consumption) from Leboncoin real estate ads and find corresponding official energy diagnostics.

## Architecture

### Source files (`src/`)

- `src/popup.js` - Main popup logic, data display, and search orchestration
- `src/api/location-client.js` - Backend API client (validation, search, Google Maps links)
- `src/utils/dom-helpers.js` - Secure DOM manipulation helpers (element creation, result list rendering)
- `src/utils/error-messages.js` - Centralized French error messages and codes
- `src/utils/score-calculator.js` - Score-to-color mapping for search results
- `src/index.js` - Re-exports all modules

### Built / extension files

- `manifest.json` - Chrome extension manifest (V3)
- `popup.html` - Extension popup UI
- `popup.js` - Built bundle (generated by `npm run build` via esbuild)

### Data Extraction Flow (src/popup.js)

1. **Primary Strategy**: Parse `__NEXT_DATA__` JSON embedded in Leboncoin pages to extract structured ad data
2. **DOM Scraping Fallback**: If data is missing, scrape the page using selectors and regex
3. **Visual Extraction**: For DPE/GES badges, analyze computed styles to find the "active" letter by comparing element sizes

### Location Search Integration

The extension calls a backend API (`POST /api/location/search`) with extracted ad data. The backend handles querying the energy performance database, scoring, and returning sorted results. Search filters include:

- Location (zipcode or city)
- DPE/GES ratings (exact match)
- Surface (+/- 10%)
- Diagnosis date (+/- 7 days)
- Primary energy consumption (+/- 10%, optional)

Results are scored based on deviation from ad data.

## Development

### Building

```bash
npm run build
```

This bundles `src/` into `popup.js` using esbuild.

### Loading the Extension

1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this directory

### Testing

```bash
npm test              # Run all tests
npm run test:unit     # Unit tests only
npm run test:integration  # Integration tests only
npm run test:e2e      # E2E tests only
```

## Key Implementation Details

- Extension only activates on Leboncoin real estate pages (URL check in `extractRealEstateData`)
- Uses `chrome.scripting.executeScript` to inject extraction logic into the active tab
- French locale throughout (labels, error messages)
- Backend API base URL configured in `src/api/location-client.js`
